# 星际公民官网导航栏插件
开发此插件的原因就是在外面看到有些人说现在的游戏官网缺乏引导，找一些页面比较繁琐，所以开发了这个插件，可以让用户在任何页面一键跳转到目标页面。并且考虑到每个人的习惯差异，插件支持自定义链接，定制自己的导航栏。

自在群友的测试反馈后，又更新了两次，主要改动就是：
1、修复了邀请页面和配件商店页面链接错误。
2、点击链接的跳转在本页面打开，不再打开新标签页，毕竟谁也不想看到浏览器里十多个标签页对吧。

## 安装方式：
1、在edge浏览器的扩展管理中搜索“RSI”，点击获取即可.
2、在github网站https://github.com/UEETianX/RSIweb-nav，下载项目文件后，根据各自浏览器的添加扩展方式安装，例如edge可以拖拽解压后的文件夹至扩展管理安装。
